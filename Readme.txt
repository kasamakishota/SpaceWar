このゲームは、２人対戦ゲームです。
移動は上下方向の移動のみです。
画面スクロールはありません。
ビームは左右方向にのみ発射できます。
ランダムで出現するアイテムを取ると、自分の飛行機が強くなったり、体力が回復したりします。
先に相手の体力をゼロにした方が勝ちです。

********キーボードの操作のコツ********
お互いに
「右手の人差し指と中指で飛行機の移動」
「左手の人差し指でビームを発射」
すると操作しやすいと思います。

********各飛行機の操作方法********

*****青（左側）の飛行機*****
「Ｗ」→　上方向へ移動
「Ａ」→　下方向へ移動
「Ｑ」→　ビーム発射

*****赤（右側）の飛行機*****
「Ｐ」→　上方向へ移動
「Ｌ」→　下方向へ移動
「Ｏ」→　ビーム発射

********アイテムの効果一覧********
赤十字マーク　→　体力回復
星マーク　　　→　３発だけ相手のビームを食らってもダメージを受けない
赤いビン　　　→　ビームのダメージが増加する
青いビン　　　→　ビームの長さが長くなる
黄のビン　　　→　ビームの幅が広くなる
緑のビン　　　→　敵機が大きくなって命中しやすくなる
紫のビン　　　→　飛行機の移動スピードが速くなる
白黒のビン　　→　ビームの速さが速くなる
